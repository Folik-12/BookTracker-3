# GitHub User Finder

**Автор:** [Ваше Имя фамилия]

### Описание

Приложение для быстрого поиска информации о пользователях GitHub и сохранения их в локальный список избранного.

### Как использовать API

Программа использует публичный эндпоинт:
`GET https://github.com{username}`
_Лимит запросов без токена: 60 в час._

### Инструкция по запуску

1. Установите зависимости: `pip install requests`
2. Запустите: `python main.py`
3. Введите никнейм и нажмите "Найти".
