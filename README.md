# Book Tracker

**Автор:** [Максим]

### Описание
Приложение для учета прочитанных книг. Позволяет хранить список литературы, указывать количество страниц и фильтровать книги по жанру (и объему более 200 страниц).

### Инструкция по запуску
1. Установите Python (версия 3.x).
2. Скачайте файл `main.py`.
3. Запустите программу командой:
   `python main.py`

### Примеры использования
- Введите данные книги "Мастер и Маргарита" и нажмите "Добавить".
- Чтобы найти толстые книги определенного жанра, введите жанр в поле фильтра и нажмите "Фильтровать" (отобразятся книги этого жанра более 200 страниц).
